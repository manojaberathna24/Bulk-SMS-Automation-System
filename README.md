This project is an educational tool designed to demonstrate how API requests and automation work.
Only the obfuscated files (sms.py and api_obf.py) are included for security reasons.

Features

Sends automated test SMS requests (ethically and legally).

Uses HTTP requests to communicate with API endpoints.

Fully obfuscated source for safe public sharing.



How TO Run

pip install -r requirements.txt

python sms.py
